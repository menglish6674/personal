Deleted BigFix Actions Forensic Audit
--------------------------------------------
Run this app to view deleted BigFix Actions.
Clicking on an Action will reveal Action Details.

This is meant to be run only when needed.

----Requirements----
#Ensure ODBC Driver 18 for SQL Server is installed
#Windows Authentiction Checkbox will only pass credentials from currently logged on user otherwise use a SQL authenticating account with at least read only access to the BFEnterprise database
