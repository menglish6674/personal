BIGSLACK v2.0

Bigslack integrates the Deployment Health Check Dashboard with SLACK and will also notify you of any new site content

This will run every 2 hours (by default).

Prerequisites:
You must create or use an existing Slack Workspace
You must create or use an existing Slack Channel
You must create a SLACK Webhook App
The endpoint that will be the target of this action should be available 24/7.  It will also need Curl installed.
